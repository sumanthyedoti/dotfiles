## Setup
- Install the Tools and Dependecies first
 - git clone this repo into `~/.dotfiles`
- run `stow .`
- Install Fonts
	- Hack (nerd)
	- Jetbrains Mono (nerd)
	- Roboto Slab
	- Roboto Mono
	- [Iosevka](https://github.com/be5invis/Iosevka)
- nvim
	- [Prerequisites](https://github.com/neovim/neovim/wiki/Building-Neovim#build-prerequisites)
	- `:Lazy Sync`
	- `:UpdateRemotePlugins`
