# Quick Look

## Keybindings

### General

| Action                     | Key Binding            |
| -------------------------- | ---------------------- |
| Open Terminal              | `Ctrl + /`             |
| Quit Neovim                | `<leader> + qq`        |
| View All Keybindings       | `<leader> + backspace` |
| Toggle Folds               | `z`                    |
| Jump to Function in Buffer | `<leader>ss`           |
| Search Keymaps             | `<leader>sk`           |

### Explorer

| Action      | Key Binding |
| ----------- | ----------- |
| Toggle Fold | `z`         |

### Buffers

| Action              | Key Binding |
| ------------------- | ----------- |
| Move Left           | `H`         |
| Move Right          | `L`         |
| Open Scratch Buffer | `<leader>.` |

### Copilot

- **Activate**: `:Copilot auth`

#### Copilot-Chat

- **Select Model**: `$`
- **Select Buffer**: `#buffer`

### Mini-ai (Part of LazyVim)

- **Function**: `f`
- **Function Attribute**: `a`
- **Any Number**: `<any-number>`
- **Space**: `<space>`
- **Quotes**: `q -> ', ",`
- **Brackets**:
  - Open Bracket: `b -> (, {, [`
  - Repeat Outer Brackets: `vababab`
